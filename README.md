tahoe-pi
========

Basic car computer for my Chevy Tahoe. Records GPS data from gpsd
and engine data from an STN1110 IC.

Currently targeting Debian/Raspbian "Wheezy" on the ARMv6 arch and Rust 0.10-pre (revision 42fc32f).
